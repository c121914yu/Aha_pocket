# Aha口袋小程序

## Hbuilder 配置
1. 建议直接暴力安装所有插件
2. tab设置为4个空格
3. 预设方案建议改成VS code,工具 -> 预设快捷方案 -> VSCode
4. 一些快捷键
	4.1 /**回车，快速生成函数注释  
	4.2 ctrl + ? 行注释  
	4.3 ctrl + shift + ? 块注释  
	4.4 ctrl + e 搜索文件  
	4.5 ctrl + f 当前文件搜索字段  
	4.6 ctrl + shift + f 全项目搜索  
	4.7 ctrl + w 关闭当前标签  
	4.8 alt + 上方123.. 切换标签
